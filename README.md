# My Portfolio

## Live Link
Check out the live site here: [https://durgasravanthip.github.io/MyPortfolio-Simple/](https://durgasravanthip.github.io/MyPortfolio-Simple/)

- Contact information

## Setup
Simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/DurgaSravanthiP/MyPortfolio-Simple.git
```
